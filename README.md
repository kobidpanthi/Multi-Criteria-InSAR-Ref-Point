# InSAR Multi Criteria Reference Point Selection Approach

This repository contains the Python scripts, Jupyter Notebooks, and MintPy configuration files developed as part of my PhD research at the University of Melbourne. The project focuses on improving the reliability of InSAR time-series analysis based on a hybrid threshold of multiple parameters followed by a multi-point spatial averaging algorithm.
